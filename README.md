# Sizwe95-Laundry
# Spin & Drop

A polished landing page for a mock on-demand laundry service in Durban CBD.

## What is included
- A modern, responsive homepage
- Service overview and pricing sections
- A booking form mock-up
- Simple styling powered by Tailwind CSS

## Run locally
Open the project directly in a browser, or serve it from a simple local web server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000.

## Publish to GitHub Pages
1. Push this repository to GitHub.
2. Open your repository settings.
3. Go to Pages.
4. Choose Deploy from a branch and select the main branch with the root folder.
5. Save and wait for the site to publish.

## Notes
- This project is a static website, so no build step or package installation is required.
- The main page is available in index.html.
